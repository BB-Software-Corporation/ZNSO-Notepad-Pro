# ZNSO-Notepad-Pro

**ZNSO Notepad Pro** —— 一个专注于写笔记的专业软件，支持 **可视化笔记**，允许使用 **图像 + 文本** 进行记录。  
这是 **BB Software Corporation** 发布的第一个开源软件。  

---

## ✨ 功能亮点
- 📝 富文本编辑：支持文字、标题、格式化。
- 🖼️ 图像笔记：可以直接插入图片，帮助可视化记录。
- 📂 工程化管理：支持项目/专辑形式保存笔记。
- 🎯 面向专业用户：提供更多高级功能（专业版）。
